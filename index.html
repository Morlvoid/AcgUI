Animation Components

轻量级动画组件库 - 白色主题简洁线条风格

v2.0.0
1

樱花飘落 (Sakura)

可配置速度和密度

使用示例

// JavaScript 方式 const sakura = new Sakura({ container: document.getElementById('container'), density: 30, // 密度 speed: 2, // 速度 color: '#ffb7c5' }).init(); // 动态更新 sakura.update({ density: 50, speed: 3 });
2

气泡上升 (Bubble)

可配置速度和密度

使用示例

// JavaScript 方式 const bubble = new Bubble({ container: document.getElementById('container'), density: 15, // 密度 speed: 1, // 速度 color: '#aae6ff' }).init(); // 动态更新 bubble.update({ density: 25, speed: 1.5 });
3

边框动画 (Border)

可配置速度和线条样式

使用示例

// JavaScript 方式 const border = new Border({ container: document.getElementById('container'), speed: 1, // 速度 lineWidth: 2, // 线条宽度 color: '#333' }).init(); // 重新播放 border.update({ speed: 1.5 });
4

星星动画 (Star)

可配置颜色和位置
★

使用示例

// JavaScript 方式 const star = new Star({ container: document.getElementById('container'), color: '#ffd700', position: { x: 50, y: 50 } // 位置百分比 }).init(); // 更新位置和颜色 star.update({ color: '#ff6b6b', position: { x: 30, y: 70 } });
5

背景图片 (Background)

可配置图片和切换间隔
背景图片将显示在这里

使用示例

// JavaScript 方式 const bg = new Background({ container: document.getElementById('container'), images: ['image1.jpg', 'image2.jpg'], duration: 5000 // 切换间隔(ms) }).init(); // 更新切换间隔 bg.update({ duration: 3000 });
6

呼吸效果 (Breathing)

可配置速度和方向

使用示例

// JavaScript 方式 const breathing = new Breathing({ element: document.getElementById('target'), speed: 1, // 速度 direction: 'vertical', // 方向 scale: 0.2 // 缩放范围 }).init(); // 更新配置 breathing.update({ speed: 1.5, direction: 'both' });
